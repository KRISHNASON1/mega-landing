# Hero Images Folder

Place your hero section images in this folder.

## File Naming Convention

Name your images as:
- `1.png` - First image
- `2.png` - Second image
- `3.png` - Third image
- `4.png` - Fourth image

You can add more images by updating the `heroImages` array in `/src/components/home/Hero.jsx`

## Supported Formats
- PNG (.png)
- JPEG (.jpg, .jpeg)
- WebP (.webp)

## Recommended Image Dimensions
- Width: 800-1200px
- Height: 800-1200px
- Aspect Ratio: Square or Portrait (1:1 or 3:4)
